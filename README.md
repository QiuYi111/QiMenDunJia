# 奇门遁甲
奇门遁甲之术乃是连接五行，八卦，九宫，天干，地支等等术数的大一统之术。可以测事，可以测命，最重要的是奇门遁甲并非告知宿命的占卜，而是以**宫干门星神**的视角帮助人们看清现实，找出破局之法。

奇门遁甲乃是中国古代哲学的结晶，展示的是一套世界观。
